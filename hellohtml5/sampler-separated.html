HTML5 and CSS3 Sampler (Separate Files)

This page presents a sampling of features from the latest HTML5 and CSS3 standards. It’s not meant to be comprehensive, but does highlight some capabilities that were impossible (or highly ugly, code-wise) in older versions of HTML and CSS.

There is one rule on this page: no images allowed. Or, images.

Also, while this might strike you as a rule, the way we operate we assume this as a matter of course: The page must validate!
